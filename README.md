# 🎮 AstrBot 永劫无间招募插件 (Naraka Recruit)

一个为 [AstrBot](https://github.com/Soulter/AstrBot) 开发的简单插件，专门用于在 QQ 群内发布《永劫无间》组队招募信息，并自动 @全体成员。

## ✨ 功能特性

- **一键招募**：支持双排、三排、娱乐模式快捷触发。
- **自动宣讲**：自动艾特全体成员（需机器人拥有管理员权限）。
- **内置冷却**：默认 30 秒指令冷却，防止刷屏。
- **每日限额**：每人每天限发 3 次招募，保护群聊环境。
- **智能识别**：支持提及机器人昵称（小劫宝）触发。

## 🚀 安装方法

1. 在 AstrBot 管理面板中，进入「插件」页面。
2. 选择「通过 GitHub 安装」，输入本仓库地址：
   `https://github.com/LK-von-Clausewitz/astrbot_plugin_naraka_recruit`
3. 安装完成后重启 AstrBot 即可。

## 🛠 使用说明

在群内输入以下指令（需输入小劫宝）：

| 指令 | 效果 |
| :--- | :--- |
| `小劫宝 三排组队` | 发起三排招募并 @全体成员 |
| `小劫宝 双排组队` | 发起双排招募并 @全体成员 |
| `小劫宝 娱乐组队` | 发起娱乐招募并 @全体成员 |

> **注意**：为了使 `@全体成员` 生效，请确保机器人已获得该群的**管理员**身份。

## ⚙️ 插件配置

你可以直接修改 `main.py` 中的以下变量来调整限制：

* `self.cooldown_seconds = 30`：调整两次招募之间的冷却秒数。
* `self.daily_limit = 3`：调整每个用户每天允许招募的次数。
* `self.bot_nickname = "小劫宝"`：修改为你机器人的名字。
