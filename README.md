# ⚔️ 永劫无间招募插件 for AstrBot

专门为永劫无间游戏玩家设计的 AstrBot 招募插件，支持双排、三排、娱乐三种模式。

## 功能特性

- 🎮 **限定模式**：只支持双排、三排、娱乐三种永劫无间常用模式
- 📢 **@全体成员**：普通成员可请求机器人代为 @全体发布招募
- 🛡️ **防滥用机制**：冷却时间 + 每日次数限制
- 💬 **精美格式**：专为永劫无间定制的消息格式

## 使用方法

```
/recruit <模式> <备注>
```

### 可用模式
- 👥 **双排** - 双人排位赛
- 👥👥 **三排** - 三人排位赛  
- 🎮 **娱乐** - 娱乐模式

### 示例
```
/recruit 三排 来个会玩的迦南
/recruit 双排 上分dd
/recruit 娱乐 快乐就完事儿了
```

## 配置说明

可在 AstrBot WebUI 中配置：
- `cooldown_seconds`：冷却时间（默认30秒）
- `daily_limit`：每日限制次数（默认3次）
