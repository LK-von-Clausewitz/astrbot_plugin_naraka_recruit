# 🎮 AstrBot 永劫无间全模式招募插件

专门为《永劫无间》玩家群设计的 AstrBot 插件。支持自动 @全体成员 发布招募，涵盖从天选、征神到暗域摸金的所有模式。

## ✨ 功能特性

- **模式全覆盖**：支持双排、三排、刀房、征神、摸金、破镜劫、地脉。
- **强制全体提醒**：使用 `AtAll` 组件，确保群员能收到系统级提示音。
- **智能识别逻辑**：完美兼容 QQ 的「点击头像艾特」与「直接输入名字」。
- **防刷屏机制**：内置冷却时间（30s）与每日使用上限。

## 🛠 指令说明

### ⚠️ 触发须知（重要！）
由于 QQ 协议限制，**直接输入 `@小劫宝` 可能会因被识别为系统节点而导致指令失效**。
建议采用以下两种方式之一：
1. **方式 A**：点击头像艾特机器人 + `三排组队`
2. **方式 B**：直接输入 `小劫宝 三排组队`

## 🚀 安装与配置

1. 在 AstrBot 管理面板中通过本仓库地址安装。
2. **权限要求**：请务必将机器人设为 **群管理员**，否则无法触发 `@全体成员`。
3. **自定义**：如需修改每日次数或机器人昵称，请编辑 `main.py` 中的 `self.daily_limit` 和 `self.bot_nickname`。
